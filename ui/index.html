Agentic Event Flow

1. Trigger

BiLSTM detects fall probability over threshold from pose windows.

2. Context

Aggregate sparse video features: pose state, motion level, recovery attempt.

3. Gemma 270 Routing

Gemma emits validated JSON actions, including alert message and incident metadata.

4. Action

Local notification appears in-app and patient log is appended to data/patient_events.jsonl.

System Features

Deterministic Safety

If model output is invalid, fallback policy still alerts and logs incident.

Low-Latency Design

Event-driven reasoning with a short re-check stage avoids impractical frame-by-frame LLM loops.

Clinical Traceability

Each incident carries timestamp, room id, severity, status, and event type for auditability.
